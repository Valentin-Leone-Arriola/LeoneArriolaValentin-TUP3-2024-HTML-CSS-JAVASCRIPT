Ejercicio 2: Etiquetas y Atributos Crea un documento HTML que contenga un formulario con los siguientes campos: un campo de texto para el nombre, un campo de correo electrónico, un campo de contraseña y un botón de envío. Asegúrate de que cada campo tenga una etiqueta apropiada y atributos correctos.






Ejercicio 3: Estilos CSS inline y en archivo externo Crea un documento HTML con un párrafo que tenga el texto "Este es un párrafo con estilo inline" y aplícale estilo inline para que el texto sea de color rojo y tenga un tamaño de fuente de 20px.
